# Arrays

This folder contains all Array related DSA problems.

## Structure
- [Array 1D](./Array_1D/)
    1) **Find the element**
        i) largest element
       ii) smallest element

    2) **Searching**
        i) Linear Search
       ii) Binary Search

    3) **Reverse an array**
        i) With extra space
       ii) Without extra space

    4) **Subarray problems**
        i) Print all subarrays
       ii) Max Subarray Sum
            a) Brute Force
            b) Optimal (Medium)
            c) Kadane's Algorithm (Most Optimal)

    5) **Buy and Sell Stocks**

    6) **Trapping Rain Water**

- [Sorting Algorithms](./Sorting_Algorithms/)

    1) **Bubble Sort** (O(n^2))
    2) **Selection Sort** (O(n^2))
    3) **Insertion Sort** (O(n^2))
    4) (**Counting Sort** (O(n+k))



- [Array 2D](./Array_2D/)
    1) **Spiral printing of matrix**
    2) **Diagonal sum**
    3) **Search a sorted matrix**
        i) Linear Search (Brute Force)
       ii) Col/Row Binary Search (Medium Optimal)
      iii) Staircase Search (Most optimal)
